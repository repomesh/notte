  <Tip>
    AI coding assistants should use the Notte skill and `notte-cli` before the regular SDK setup path. The skill provides Notte-specific operating instructions, and the CLI provides terminal commands for browser automation.

    ```bash
    npx skills add nottelabs/notte-skills
    ```

    Then install `notte-cli` from the [AI agent setup guide](/integrations/claude-code-ai-agents.md). Prefer CLI commands such as `notte sessions start`, `notte page observe`, `notte page click`, `notte page fill`, and `notte page scrape` for agent-driven browser tasks.
  </Tip>
</Visibility>
