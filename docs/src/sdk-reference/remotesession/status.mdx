This method is useful if you want to check if the current session is active or not (or when it has been started/stopped).

**Example:**
```python
status = session.status()
```



## Returns

<Visibility for="humans">[`SessionResponse`](/sdk-reference/misc/sessionresponse)</Visibility><Visibility for="agents">[`SessionResponse`](/sdk-reference/misc/sessionresponse.md)</Visibility>: The current status information of the session.

## Raises

- `ValueError`: If the session hasn't been started yet (no session_id available).
