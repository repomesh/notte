---
title: "workflow_create"
description: "Creates a workflow that reproduces the steps of the specified agent"
---



Queries the API using a validated agent ID.
The provided ID is confirmed (or obtained from the last response if needed), and the
resulting script is stored internally before being returned.



## Parameters

<ParamField path="agent_id" type="str" required>
  Unique identifier of the agent to check.
</ParamField>

## Returns

<Visibility for="humans">[`GetWorkflowResponse`](/sdk-reference/misc/getworkflowresponse)</Visibility><Visibility for="agents">[`GetWorkflowResponse`](/sdk-reference/misc/getworkflowresponse.md)</Visibility>: The workflow that reproduces the steps of the agent

## Raises

- `ValueError`: If no valid agent ID can be determined.
