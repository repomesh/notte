---
title: "replay"
description: "Get presigned URLs for the specified agent's session replay"
---

<Warning>
  This method is deprecated. Use <Visibility for="humans">[`session.replay()`](/sdk-reference/remotesession/replay)</Visibility><Visibility for="agents">[`session.replay()`](/sdk-reference/remotesession/replay.md)</Visibility> instead.
</Warning>

```python
# Deprecated — prefer session.replay()
replay = client.agents.replay(agent_id="your-agent-id")
replay.download("replay.mp4")
```



## Parameters

<ParamField path="agent_id" type="str" required>
  The identifier of the agent to get the replay for.
</ParamField>

<ParamField path="wait" type="bool" default="True">
  If `True`, polls until the replay is ready instead of raising on 404.
</ParamField>

<ParamField path="timeout" type="float" default="240.0">
  Maximum seconds to wait for the replay to become available.
</ParamField>

<ParamField path="poll_interval" type="float" default="5.0">
  Seconds between polling attempts.
</ParamField>

## Returns

<Visibility for="humans">[`ReplayResponse`](/sdk-reference/misc/replayresponse)</Visibility><Visibility for="agents">[`ReplayResponse`](/sdk-reference/misc/replayresponse.md)</Visibility>: Contains presigned URLs for MP4 download and HLS playlist.
