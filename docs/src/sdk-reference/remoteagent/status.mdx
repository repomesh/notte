This method retrieves the current state of the agent, including its progress,
actions taken, and any errors or messages.

```python
status = agent.status()
```




## Returns

<Visibility for="humans">[`LegacyAgentStatusResponse`](/sdk-reference/misc/legacyagentstatusresponse)</Visibility><Visibility for="agents">[`LegacyAgentStatusResponse`](/sdk-reference/misc/legacyagentstatusresponse.md)</Visibility>: The current status of the agent execution.

## Raises

- `ValueError`: If the agent hasn't been run yet (no agent_id available).
